# Sitemap Concept: uXPRT

## Sitemap Structure

This outlines the basic page hierarchy for the uXPRT application.

- **Home/Dashboard**
  - Task Overview
  - Recent Artifacts
  - Initiate New Task
- **Chat**
  - Conversation with Agent(s)
  - Agent Selection/Switching
- **Artifacts Library**
  - Search and Filter
  - List/Grid of Artifacts
  - Artifact Details (View/Download/Manage)
- **Settings**
  - User Profile
  - Account Settings
  - Notifications
  - Integrations (Future)
